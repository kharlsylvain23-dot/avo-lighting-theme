# Sections Overview

This folder contains modular Shopify Online Store 2.0 sections for a single-product lamp store. Each section is a Liquid file with JSON schema for app block support and theme editor compatibility.

## Sections
- header.liquid
- hero.liquid
- transformation-steps.liquid
- alternating-content-blocks.liquid
- benefits.liquid
- features.liquid
- social-proof.liquid
- comparison-table.liquid
- faq.liquid
- product-showcase.liquid
- footer.liquid

All sections are designed for a minimalist, classy look (black, white, gray, yellow accents), and are responsive for mobile and desktop.
